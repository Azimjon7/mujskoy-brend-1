MUJSKOY BREND STARTER SHOP

NIMALAR QILINDI:
- Eski demo mahsulotlar va product ma'lumotlari saytdan olib tashlandi.
- Bosh sahifa, do'kon, product detail, savat, checkout va admin panel qayta ulandi.
- Backend qo'shildi (Node.js + Express).
- Mahsulotlar JSON faylga saqlanadi.
- Rasmlar uploads/ papkaga yuklanadi.
- Telefon orqali ham admin.html ochib mahsulot yuklash mumkin.

QANDAY ISHLATILADI:
1) Node.js o'rnating.
2) Terminalda shu papkaga kiring.
3) npm install
4) npm start
5) Brauzerda: http://localhost:3000
